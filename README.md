# 🗺 GoRoute
> 졸업작품 | 2025.03 ~ 2025.06

React Native 기반 외국인 대상 한국 관광 길찾기 애플리케이션입니다.
독일 교환학생 경험을 바탕으로, 외국인이 한국에서 관광지를
쉽게 찾고 이동할 수 있도록 돕기 위해 기획했습니다.

> 추후 백엔드 구현 및 디벨롭을 통해 App Store 출시를 목표로 하고 있습니다.

## ✨ 주요 기능
- 📍 Naver Map API 기반 관광지 위치 확인 및 길찾기
- 🔎 관광지 검색 기능
- 🗺 지도 기반 주변 관광지 정보 제공
- 🌐 외국인 친화적 UI/UX 설계

## 🛠 기술 스택
| 구분 | 기술 |
|------|------|
| Frontend | React Native |
| 개발환경 | Expo |
| Map | Naver Map API |

## 💡 기획 배경
2024~2025년 독일 힐데스하임 대학교 교환학생 프로그램 참여 중
외국인으로서 낯선 환경에서 관광지를 찾는 어려움을 직접 경험하고,
귀국 후 이를 해결하기 위한 서비스로 기획했습니다.

## 👩‍💻 담당 역할 (1인 개발)
- 서비스 기획 및 UI/UX 설계
- Naver Map API 연동 및 지도 기능 구현
- 관광지 정보 제공 화면 개발
